Billing Report Generator

Report Format

This report generates an Excel file with the following columns:

  • Date - Service date
  • Start Time - Time started
  • End Time - Time completed
  • Total Min - Total minutes worked
  • HR $ - Cost based on hourly rate ($150/hr)
  • Site - Property location
  • Qty Salt (yrd) - Bulk salt quantity in yards

The report includes a header with total time and truck rate information.
